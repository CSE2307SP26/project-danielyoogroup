## User stories

1. A bank customer should be able to deposit into an existing account. (Shook)
2. A bank customer should be able to withdraw from an account. (Anne)
3. A bank customer should be able to check their account balance. (Anne)
4. A bank customer should be able to view their transaction history for an account. (Marc)
5. A bank customer should be able to create an additional account with the bank. (Marc)
6. A bank customer should be able to close an existing account. (Shelby)
7. A bank customer should be able to transfer money from one account to another. (Shelby)  
8. A bank adminstrator should be able to collect fees from existing accounts when necessary. (Daniel)
9. A bank adminstrator should be able to add an interest payment to an existing account when necessary. (Daniel)
